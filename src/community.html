Our community

a.k.a. our greatest feature!

One of the stand out features of the Aristotle Metadata Registry is that we are building a growing community of developers, translators, writers and quality assurers all who love and support open source software and are passionate about building systems that make data make sense.

And we want you to join us!

We've gone out of our way to make sure that Aristotle is open and acccessible ‡ and easy to use, install and develop because we believe that to make a system that truly supports an open standard requires an open source solution.

Join in and help make history!

Our code is hosted on Github and we've done everything we can to make it easy to start developing new extensions, adding new translations or reviewing documentation. Just like the parable of stone soup, every little bit counts, so it doesn't matter if you are a veteran in your field, starting out or somewhere in between you are welcome to join us! {% comment %} As an added incentive, we frequently reward new developers who help us tackle especially tricky or interesting features and bugs ! {% endcomment %}

If you get stuck or want to find out what we are talking about find us online using any of the ways listed below.
